# Summary
Converts [SQLAlchemy](https://www.sqlalchemy.org/) models to [AtlasGo](https://atlasgo.io/) HCL.
# installation
    - pip install sqlalchemy2atlas
# Quickstart

    <!-- Insall deps? Docker? -->
    - Run `make`

    <!-- Usage -->
    - `sqlalchemy2atlasgo -h`

# Driver Support
- [x] PostgreSQL
    - [x] 14.x
- [ ] Spanner
- [ ] MySQL
- [ ] MariaDB
- [ ] SQLLite
